XmlWhitespaceStrippedSource Class

A source that is obtained from a different source by removing all empty text nodes and removing all characters XML considers whitespace at the start and end of the text content of the non-empty ones.

Syntax

public class XmlWhitespaceStrippedSource : DOMSource

Remarks

If you only want to remove text nodes consisting solely of whitespace (AKA element content whitespace) but leave all other text nodes alone you should use ElementContentWhitespaceStrippedSource instead.

Unlike Org.XmlUnit.Input.WhitespaceStrippedSource this uses XML's idea of whitespace rather than the more extensive set considered whitespace by Unicode.

since XMLUnit 2.10.0
