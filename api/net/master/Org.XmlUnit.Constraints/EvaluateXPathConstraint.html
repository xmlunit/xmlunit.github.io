EvaluateXPathConstraint Class

This NUnit3 Constraint verifies whether the evaluation of the provided XPath expression corresponds to the value constraint specified for the provided input XML object.

Syntax

public class EvaluateXPathConstraint : NUnit.Framework.Constraints.Constraint

Remarks

All types which are supported by Input.From(object)} can be used as input for the XML object against the constraint is evaluated.

since XMLUnit 2.1.0

Example

Simple Example
Example
            final String xml = "<a><b attr=\"abc\"></b></a>";
            Assert.That(xml, EvaluateXPathConstraint.HasXPath("//a/b/@attr", Is.EqualTo("abc")));
            Assert.That(xml, EvaluateXPathConstraint.HasXPath("count(//a/b/c)", Is.EqualTo("0")));

Example

Example with namespace mapping
Example
            string xml = "<?xml version=\"1.0\" encoding=\"UTF-8\"?>" +
            "<feed xmlns=\"http://www.w3.org/2005/Atom\">" +
            "   <title>title</title>" +
            "   <entry>" +
            "       <title>title1</title>" +
            "       <id>id1</id>" +
            "   </entry>" +
            "</feed>";
            var prefix2Uri = new Dictionary<string, string>();
            prefix2Uri["atom"] = "http://www.w3.org/2005/Atom";
            Assert.That(xml,
            EvaluateXPathConstraint.HasXPath("//atom:feed/atom:entry/atom:id/text()",
            Is.EqualTo("id1"))
            .WithNamespaceContext(prefix2Uri));
